AI Food Recipe Generator

An AI-powered web application that helps users generate delicious recipes instantly. Simply enter the ingredients you have, choose your dietary preferences or cuisine, and let AI create a complete recipe with step-by-step cooking instructions.

✨ Features
🤖 AI-generated recipes
🥗 Ingredient-based recipe suggestions
🌍 Multiple cuisine options
🥬 Dietary filters (Vegetarian, Vegan, Gluten-Free, etc.)
⏱️ Cooking time estimation
📋 Step-by-step cooking instructions
💡 Recipe tips and serving suggestions
📱 Responsive and user-friendly interface
🛠️ Tech Stack
Frontend: React.js
Backend: Node.js + Express.js
AI API: OpenAI/Gemini API
Database: MongoDB
Styling: CSS / Tailwind CSS / Bootstrap
🚀 How It Works
Enter available ingredients.
Select cuisine or dietary preferences.
Click Generate Recipe.
AI creates a unique recipe with ingredients, instructions, cooking time, and serving size.
